## Перевод ДНК в РНК и синтез белка

Эта программа на Python принимает на вход последовательность ДНК, переводит её в соответствующую последовательность РНК, а затем предсказывает аминокислотную последовательность, которая образуется в процессе синтеза белка. Программа также вычисляет молярную массу полученного белка и определяет процентное содержание полярных и неполярных аминокислот.

### Возможности

- **Перевод ДНК в РНК:** Программа переводит введённую последовательность ДНК в соответствующую последовательность РНК, используя следующие правила замены нуклеотидов:
  - `G` → `C`
  - `A` → `U`
  - `T` → `A`
  - `C` → `G`
  
- **Обработка ошибок:** Программа проверяет наличие недопустимых символов в последовательности ДНК и выводит сообщение об ошибке, если такие символы найдены.

- **Определение кодонов:**
  - Поиск старт-кодона (`AUG`) в последовательности РНК.
  - Определение стоп-кодонов (`UAA`, `UAG`, `UGA`), чтобы определить, где заканчивается синтез белка.
  - Проверка наличия необходимых стартовых и стоп-кодонов для корректного перевода.

- **Синтез белка:**
  - Переводит последовательность РНК в аминокислотную последовательность, начиная с первого старт-кодона (`AUG`) и заканчивая ближайшим стоп-кодоном.
  - Вычисляет молярную массу полученного белка.
  - Определяет процентное содержание полярных и неполярных аминокислот в белке.
